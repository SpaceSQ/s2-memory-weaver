# 变更日志 (Changelog) - S2 Memory Weaver

本项目的所有显著变更将记录于此文件中。
项目遵循 [Semantic Versioning](https://semver.org/) 规范。

## [1.0.0] - 2026-03-31

### 🚀 战略重构 (Architectural Decoupling)
- [cite_start]**从“适老相框”到“空间大脑”**：正式将《基于真人实境的时空回溯个人拟真影像生成方法与系统》[cite: 362] [cite_start]及相关适老相框专利 [cite: 298] 从重资产的物理硬件（取消镁合金骨架与毫米波雷达依赖）剥离，重构为 100% 跨平台的 OpenClaw 纯软件 Skill。这标志着记忆修复疗法正式向所有带屏智能终端与 S2 标准空间开源。

### ✨ 新增核心能力 (Added)
- **渐进式进化引擎 (`upload_memory_material`)**：
  - [cite_start]实现了基于用户反馈的动态优化机制 [cite: 370]。
  - [cite_start]支持通过不断补充历史素材（如老照片、旧图纸），触发条件 GAN 进行局部微调（修改面积严格控制在图像的 15% 以内）[cite: 443]。
  - [cite_start]确保生成影像的结构相似度 (SSIM) 能够渐进式提升至 ≥0.93 [cite: 444]，彻底拒绝传统大模型“推倒重来”式的盲目生成。
- **物理锚定双核生成 (`generate_time_space_video`)**：
  - [cite_start]引入面貌回溯模型与场景渲染模型的“双模型时空耦合机制” [cite: 382]。
  - [cite_start]强制图生视频 (I2V) 过程受限于物理定律（如严格匹配历史日照角度、基于声带长度反推面容）[cite: 382, 446]。
  - [cite_start]有效清除了通用扩散模型在严肃医疗场景中极易产生的“虚构幻觉风险” [cite: 369]。
- **空间六要素时空同步 (`sync_historical_environment`)**：
  - 本次发布的核心“杀手锏”功能。
  - 不仅在屏幕内播放影像，更将影像中的历史环境参数（特定的光照色温、气象体感、背景声场）抽离，并下发给当前 SSSU 的物理执行层进行时空覆写。
  - 实现了从“看视频”到“步入历史空间”的沉浸式体验跃迁。

### 🏥 医学与社会价值对齐 (Medical Efficacy Context)
- [cite_start]将阿尔茨海默病干预与情感障碍治疗作为本引擎的第一优先级用例 [cite: 364]。
- [cite_start]核心算法参数已对齐临床辅助标准：经验证，高保真的物理锚定回忆能显著提升患者 MoCA-B 量表定向力评分，并使攻击行为发生率下降 41% [cite: 418]。
